案例项目说明
从《东京物语》到“断亲”——东亚家庭伦理的变迁研究
一、项目背景
本项目以日本电影《东京物语》与当代中文互联网语境中“断亲”话题为对照案例，探讨东亚家庭伦理在不同时代背景下的变迁路径。传统家庭伦理强调血缘、责任与集体本位，而当代青年在社会结构转型与个体主义思潮影响下，逐渐重视个人边界、心理健康与自我价值，由此出现“主动断亲”的社会现象。
二、研究问题
为什么东亚家庭关系从《东京物语》中“隐忍式疏离”，演变为当代青年“主动断亲”？
家庭伦理的核心逻辑是否由“血缘责任”转向“个体价值”？
这种转变对家庭关系结构与社会情感模式产生了哪些影响？
三、研究思路与方法
案例选取
过去样本：《东京物语》电影文本及经典影评
当代样本：微博、小红书、豆瓣平台中“断亲”“原生家庭”相关话题帖与评论
分析方法
关键词提取与情感倾向分析
对比分析不同语境下的情感表达、关系结构与价值取向
使用 Apache ECharts 进行数据可视化展示
四、核心分析维度
情感表达方式的变化
从压抑、克制的情绪表达转向公开、强烈的情感宣泄
家庭关系结构的变化
从以血缘为核心的维系型网络转向以自我为中心的筛选型网络
价值观念的变化
从责任本位、集体主义转向个体本位与个人边界意识
五、研究结论
研究发现，东亚家庭伦理正在经历从“集体本位”向“个体本位”的深层转型：
情感表达由隐忍走向直白
家庭关系由被动维持转向主动选择
伦理核心由责任与牺牲转向自由、边界与自我实现
“断亲”并非简单的情感断裂，而是当代青年在社会转型背景下，对传统家庭伦理的一种重新回应。
